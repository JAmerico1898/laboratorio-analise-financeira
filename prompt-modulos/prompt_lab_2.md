<contexto>
O curso de "Análise de Demonstrações Financeiras" tem 15 aulas, de 90 minutos cada. O curso é para uma turma de graduação, formada por alunos sem experiência profissional no assunto.
A segunda <aula> é sobre "Estrutura e Lógica das Demonstrações Financeiras". 
Para cada uma das 15 aulas do curso, há um conjunto de Exercícios que servem para auxiliar a fixação do conteúdo para os alunos. O módulo 2 do aplicativo desenvolve os Exercícios relacionados à aula 2, conforme abaixo.
Aula 2: Exercícios
1. Identificar e compreender cada demonstração financeira (Balanço Patrimonial, DRE, DFC, Notas Explicativas)
2. Entender a lógica econômica das contas, mapeando eventos econômicos simples (ex.: venda a prazo, financiamento bancário)
3. Identificar onde um mesmo fato aparece em mais de uma demonstraçõe financeira.
4. Estabelecer as conexões entre as demonstrações financeiras.

Os exercícios devem seguir uma estrutura de árvores decisórias com 3 ramificações e, ao final, 3 cenários distintos. O documento anexo é um exemplo. Ele pertence a outro projeto (Laboratório de Derivativos) e foi elaborado segundo essas premissas. Observe cuidadosamente a estrutura e estética do documento anexo para segui-la. Uma observação importante é oferecer anternativas de tamanho e estrutura similares para que o aluno não identifique a melhor alternativa por sua forma, mas por sua essência.
</contexto>
