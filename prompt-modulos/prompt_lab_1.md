O curso de "Análise de Demonstrações Financeiras" tem 15 aulas, de 90 minutos cada. O curso é para uma turma de graduação, formada por alunos sem experiência profissional no assunto.
A primeira <aula> é sobre "Introdução à Análise de Demonstrações Financeiras"
Para cada uma das 15 aulas do curso, há um conjunto de Exercícios que servem para auxiliar a fixação do conteúdo para os alunos. O módulo 1 do aplicativo desenvolve os Exercícios relacionados à aula 1, conforme abaixo.
Aula 1: Exercícios
1. “Quais decisões um investidor, um banco e um gestor tomam a partir das mesmas demonstrações?”
2. Classificar decisões empresariais segundo o usuário da informação
3. Decisões suportadas pela análise financeira
4. Limitações da informação contábil (estimativas, julgamento, conservadorismo)
Os exercícios devem seguir uma estrutura de árvores decisórias com 3 ramificações e, ao final, 3 cenários distintos. O documento anexo é um exemplo. Ele pertence a outro projeto (Laboratório de Derivativos) e foi elaborado segundo essas premissas. Observe cuidadosamente a estrutura e estética do documento anexo para segui-la.
